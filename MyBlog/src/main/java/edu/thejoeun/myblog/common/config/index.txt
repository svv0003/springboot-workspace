구현할 기능
1) 전일(24H/종가/평균) 대비 현재가 등락률
   전일 기준가(미정) 대비 현재가 차이를 표시한다.
   가격 클릭 시 (2)번 지도에 표시되는 유종으로 선택된다.

2) 지역별 최저가 지도 표시
   (1)번에서 선택한 유종을 기준으로 조회한 지역별 현재 최저가를 지도상에 표시한다.
   전국 평균가를 기준으로 높거나 낮은 지역은 색상으로 구분한다.
   지역 클릭 시 (3)번 그래프에 표시되는 지역으로 선택된다.

3) 유가추이 그래프
   (2)번에서 선택한 지역과 전국의 시세 변동 그래프를 표시한다.
   조회되는 데이터는 기간 (1주, 1달, 전체) 선택하도록 한다.

[모바일]
1) 화면 하단에 티커를 생성하여 3초마다 유종 데이터가 전환된다.
2) (1)번 유종 선택할 수 없으므로 화면 하단에 유종 버튼을 따로 생성하여 클릭하도록 설정한다.
3) (2)번 지도 선택할 수 없으므로 옵션으로 지역 선택하도록 한다.
